新增專案

從上一節安裝 BuckleScript 後，你可以產生一個輕量專案範本：

bsb -init my-new-project

或者，開始新的 Reason 專案：

bsb -init my-new-project -theme basic-reason

編譯 & 執行專案：

cd my-new-project
npm run build
node src/demo.bs.js

就這樣！ 剛剛已經編譯且執行過 JS 結果。 歡迎檢閱 Js 輸出，新增幾個檔案到 my-new-project/src，等等。 希望我們有交付如我們承諾的精實工具鍊及整潔的輸出 =)。

稍候的章節我們將解釋 bsb 上述 npm run build 所呼叫到的建置系統。
