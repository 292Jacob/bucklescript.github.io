Compatibilidad con navegadores y Polyfills

BuckleScript compila a JavaScript ES5, con la excepción de permitir la compilación al módulo ES6 importar/exportar.

Para navegadores anteriores, además necesitarás hace un polyfill a TypedArray. Las siguientes funciones de la librería estándar OCaml lo requieren:

  • Int64.float_of_bits
  • Int64.bits_of_float
  • Int32.float_of_bits
  • Int32.bits_of_float

Si no usas estas funciones, estás bien. De otra manera, será una error en el tiempo de ejecución.
