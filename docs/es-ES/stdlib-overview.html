Resumen

BuckleScript es en su mayoria OCaml, asi que comparten la misma biblioteca estandar. La version de OCaml sintaxis esaqui. La version de razon sintaxis esaqui.

Notaque BuckeScript esta actualmente en OCaml v4.02.3. vamos a actualizar a un OCaml mas nuevo mas tarde.

Ademas, ofrecemos unos modulos adicionales, documentadosaqui:

  • Dom: contiene tipos de DOM. El DOM es muy dificil de enlazar, y hemos decidido mantener los tipos de stdlib solamente y que los usuarios puedan enlazar con el subconjunto de DOM ellos necesitan rio abajo.
  • Nodopara nodo-especifico APIs. Experimental;! contribucion bienvenida!
  • jstodas las APIs de JS y modulos familiar estan aqui! Por ejemplo si usted quiere usar elJS Array APIsobre laOCaml Array APIporque usted esta mas familiarizado con la anterior,adelante.
