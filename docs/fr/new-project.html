Nouveau Projet

Une fois que vous avez installé BuckleScript grâce à la section précédente, vous pouvez générer un modèle de projet léger :

bsb -init my-new-project

Autrement, pour demarrer un noveau projet Reason :

bsb -init my-new-project -theme basic-reason

Pour compiler et excuter le projet que vous venez de créer:

cd my-new-project
npm run build
node src/demo.bs.js

Et c’est tout ! Vous venez juste de contruire une fois votre prjet et vu l'output JS. N'hesitez pas à etudier l'output JS, ajouter des nouveaux fichiers à my-new-project/src, etc. Hopefully we've delivered on our promise of being a lean toolchain with clean output =).

We'll explain bsb, the build system called by the above npm run build, in a later section.
